# HabitTracker
Трекер привычек. Приложение для отслеживания и формирования привычек с визуализацией прогресса и системой напоминаний

## 1.Состав команды
Величаева София 

## 2.Проблема: Людям сложно формировать новые привычки из-за отсутствия контроля и возможности отслеживать их выполнения
Решение: Веб-приложение, которое помогает создавать привычки, выполнять их и отслеживать выполнение, получать email-напоминания и визуализировать прогресс через календарь

## 3.Стек
Backend: Python 3.14, Django 6.0.3 
Frontend: HTML5, CSS3, Bootstrap 5 
База данных: SQLite 
Дополнительно: GitHub, SMTP, Django Management Commands, Django Signals 


## 4.Основные возможности:
- Регистрация и авторизация пользователей
- Создание, редактирование и удаление привычек
- Отметка выполнения с привязкой к дате
- Визуализация прогресса (календарь)
- Профиль пользователя с фото 
- Страница текущего дня
- Наличие базовых привычек
- Просмотр привычек по категориям
- Напоминания на почту


## 5.Планы на будущее:
- Система достижений и уровней (награды)
- Графики и статистика прогресса
- Мотивирующие цитаты на каждый день
- Конфетти при выполнении привычки (визуальная красота)
