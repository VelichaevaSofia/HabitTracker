# HabitTracker
Трекер привычек. Приложение для отслеживания и формирования привычек с визуализацией прогресса и системой мотивации
 
 
 Основные возможности:
- Регистрация и авторизация пользователей
- Создание, редактирование и удаление привычек
- Отметка выполнения с привязкой к дате
- Визуализация прогресса (графики, календарь)
- Личный кабинет
- Страница текущего дня
- Базовые привычки для новичков
- Напоминания на почту
